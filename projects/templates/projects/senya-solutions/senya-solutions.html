The Website

I used Astro as my SSG (Static Site Generator). This was because it is a simple, fast, and easy to use SSG. Furthermore, it has support for using other frameworks such as React, these components are then partially hydrated on the client side. This allows me to ship minimal javascript to the client, and hence, have a fast website.

TailwindCSS was used as my CSS framework. This was because it is a utility first CSS framework, which allows me to easily create custom components. It also has a large number of components that I can use, and it is easy to customise. I initally didn't think I would like it as I was more used to frameworks like Chakra UI, however I have grown to love it.
